# Safari Omnibar-like keyword search

This extension provides implements a keyword search from Safari's url bar.

## Example

`am ipod touch` will search for `ipod touch` directly in Amazon's website.

## Bundled Site Keywords

- Amazon: `amazon`
- Wikipedia: `wiki`
- Ebay: `eb`
- Newegg: `ne`
- Duckduckgo: `ddg`
- YouTube: `yt`
- Rotten Tomatoes: `rt`
- Wolfram Alpha: `wa`
- Zappos: `za`
- Twitter: `tw`
